Nombre de la app: FGC Fighting Game Coach
Descripción: FGC es una aplicación de teléfono para gamers aficionados a videojuegos de pelea
 que les permite mejorar sus habilidades y ver su progreso a través de rutinas y objetivos.

Link al repositorio: https://github.com/20970594/FGC-Fighting_Game_Coach

Link al video en Youtube: https://youtu.be/MVvnht6-HBU

•	Requerimientos funcionales:
o	El sistema debe permitir al usuario cambiar de videojuego.
o	El sistema debe permitir al usuario crear, editar, compartir, visualizar, comentar y puntuar guías de estrategias.
o	El sistema debe permitir al usuario crear, editar, eliminar, compartir, visualizar, comentar y puntuar rutinas de los juegos.
o	El sistema debe permitir al usuario ver los datos y estadísticas de los personajes de los juegos.
o	El sistema debe permitir al usuario vincular sus cuentas de los juegos.
o	El sistema debe permitir al usuario ver y mostrar su estado de conexión y rango del juego.
o	El sistema debe permitir al usuario ver su progreso de rutinas creadas, completadas y en progreso.
o	El sistema debe gestionar las paginas de estrategias, rutinas, personajes y perfil según el videojuego seleccionado.

•	Requerimientos no funcionales:
o	El sistema debe tener una interfaz de fácil uso y navegación.
o	El sistema debe permitir una transición rápida entre páginas y videojuegos.
o	El sistema debe estar siempre disponible.
